MTB Trail Finder

MTB Trail Finder allows you to search for mountain biking trails based on a user-provided location and save any trail to your trail favorites list. Each returned trail contains trail length, a description and directions. I built this app because as a new mountain biker, I wanted to be able to easily and quickly search for mountain biking trails in my area and when traveling. This is my personal favorite project as this was my first fully functional full-stack app. I was responsible for all aspects of this app.
